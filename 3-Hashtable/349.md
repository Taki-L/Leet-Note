# [349. Intersection of Two Arrays](https://leetcode-cn.com/problems/intersection-of-two-arrays/)

Given two integer arrays nums1 and nums2, return an array of their intersection. Each element in the result must be unique and you may return the result in any order.

Â 

**Example 1:**
```c++
Input: nums1 = [1,2,2,1], nums2 = [2,2]
Output: [2]
```
**Example 2:**
```c++
Input: nums1 = [4,9,5], nums2 = [9,4,9,8,4]
Output: [9,4]
Explanation: [4,9] is also accepted.
```

**Constraints:**

1 <= nums1.length, nums2.length <= 1000

0 <= nums1[i], nums2[i] <= 1000

# Solution1-Violent_1
```c++
class Solution {
public:
    bool is_included(int num, vector<int>& vec){
       for(int i = 0; i < vec.size(); i++){
           if(num == vec[i]) return true;
       }
       return false;
    }
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        vector<int> vec_save = {};
        for(int j = 0; j < nums1.size();j++){
            if(is_included(nums1[j],nums2)&&!(is_included(nums1[j],vec_save)))
                vec_save.push_back(nums1[j]);
        }
        return vec_save;
    }
};
```

## Idea
* It's a direct but a bit time-consuming solution
* Create a vector named `vec_save` to save each element repeated in two vectors
* For the every element in `nums1`, check if it is shown in 'nums2', through the function `is_included` to achieve
* Before push it back to the `vec_save`, check whether it has appended in the vector

# Solution2-Violent_2
```c++
class Solution {
public:
    bool isnot_included(int num, vector<int>& vec){
       for(int i = 0; i < vec.size(); i++){
           if(num == vec[i]) return false;
       }
       return true;
    }
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        sort(nums1.begin(), nums1.end());
        nums1.erase(unique(nums1.begin(), nums1.end()), nums1.end());
        for(int j = 0; j < nums1.size();j++){
            if(isnot_included(nums1[j],nums2)) {
                nums1.erase(nums1.begin()+j);
                j--;
            }
        }
        return nums1;
    }
};
```

## Idea

* The previous solution creates a new vector to save, in this solution, I operate directly on `nums1`
* That needs to remove the duplicate elements in `nums1`, so I use `sort` function to sort `nums1`, and then use `erase` and `unique` function to remove the duplicate elements
* Here it's a little different from before, it's not `is_included`, instead, it's `isnot_included`, to remove every element from `nums1` which is not shown in `nums2`
* When removing, I use `erase`, which will cause the length of the vector reduced by one, so here is a `j--`, moving the pointer to the right position
