# [242. Valid Anagram](https://leetcode-cn.com/problems/valid-anagram/)

Given two strings s and t, return true if t is an anagram of s, and false otherwise.

 

**Example 1:**
```c++
Input: s = "anagram", t = "nagaram"
Output: true
```

**Example 2:**
```c++
Input: s = "rat", t = "car"
Output: false
```

**Constraints:**

1 <= s.length, t.length <= 5 * 104

s and t consist of lowercase English letters.
 

**Follow up:**

What if the inputs contain Unicode characters? How would you adapt your solution to such a case?

# Solution1-Array
```c++
class Solution {
public:
    int strA[26] = {0};
    int strB[26] = {0};
    void countString(string ss,bool flag){
        int slen = ss.length();
        const char* str;
        str = ss.c_str();
        if(flag) for(int i = 0; i < slen; i++) strA[str[i]-'a'] += 1;
        else for(int j = 0; j < slen; j++) strB[str[j]-'a'] += 1;
    }
    bool isAnagram(string s, string t) {
        countString(s,true);
        countString(t,false);
        for(int k = 0; k < 26; k++){
            if(strA[k]!=strB[k]) return false;
        }
        return true;
    }
};
```

## Idea

* Before I learn Hashtable, I used the knowledge of array to solve this problem
* When define an array, the array name is the pointer address of the array header element, so first I use a `int*` function, wanted to return a array to count, but the local pointers will be destroyed on retrun
* So I defined two arrays at the very first to deal with this.
